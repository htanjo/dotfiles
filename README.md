# dotfiles
> My personal dotfiles for Mac.

## Requirements
- vim
- oh-my-zsh
- ranger
- autojump
- nvm

## Setup
```
$ git clone --recursive https://github.com/htanjo/dotfiles.git ~/dotfiles
$ sh ~/dotfiles/install.sh
```
